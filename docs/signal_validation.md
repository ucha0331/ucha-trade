## VAL-001: IS/OOS検証・7銘柄ポートフォリオ → 条件付き合格（2026-07-14）

### 検証設計

- **手法**: 約2年（487営業日）を時系列で前半60%（IS: 2024-09-11〜2025-10-17）/ 後半40%（OOS: 2025-10-20〜2026-07-14）に分割
- **パラメータ**: 現行凍結値を全銘柄共通で使用（保有5日・両方・損切り-3%・ADXフィルターON）。**銘柄別最適化はしない**（FX v4プロトコルと同じ理由：やれば7銘柄分の過学習になる）
- **合格基準（FX検証計画から流用）**: IS PF > 1.3 → OOS PF > 1.1 維持
- **⚠️ データ汚染申告**: ADXフィルター・急騰ガードはKDDI/トヨタの直近データを見て設計したため、この2銘柄は汚染済み・参考値。**未使用5銘柄（MUFG・ソニー・郵船・第一三共・商事）が真のOOS**（FX v4の「未使用ペアが真のOOS」と同じ構造）
- **残る限界**: 銘柄は未使用でも、OOS期間（2025-10以降）はフィルター設計時に観察していた相場局面と重なる。完全に無垢なのは「未使用銘柄×IS期間」のみ

### 判定: 条件付き合格

**FXとの決定的な違い**: FX v4はOOSでPF 0.77に崩壊し撤退した。株側は**OOSでPF 1.17〜1.23を維持**し、事前宣言基準（>1.1）を上回った。エッジは実在する。

**ただし以下の限定を明記する**:

1. **エッジは薄い**: IS→OOSで期待値が1/3に減衰（+0.70→+0.27%/回）。IS成績の額面を将来に期待してはいけない
2. **コスト感応度**: 往復コスト0.1%なら真のOOS期待値+0.12%/回で生存、0.2%なら+0.02%でほぼ消滅。ゼロ手数料帯（国内現物）＋流動性の高い大型株に限定する運用が前提条件
3. **銘柄分散が激しい**: OOS PFはソニー0.40・第一三共0.34〜商事2.60・MUFG3.41。単一銘柄への集中適用は危険。ソニー・第一三共の共通敗因は「一方向にじり続ける相場での逆張り買いの連続損切り」
4. **真の付加価値はリターンよりリスク圧縮**: OOS合計リターンは戦略+42.0% vs B&H+39.7%でほぼ同等。しかし最大DDは**全7銘柄で例外なくB&Hより浅い**（15〜86%、中央値39%に圧縮）。「同等のリターンを、半分以下の痛みで」がこのエンジンの実像

### 実運用への反映（事前宣言）

- 現行運用（シグナルに従う・ATR損切り厳守・週次レポート）を**継続**。エンジンのパラメータは凍結し、次の見直しは実トレード30回蓄積後
- 1銘柄への資金集中はしない（銘柄分散が激しいため）
- 「過去成績が良い銘柄を選んで適用する」ことはデータマイニングになるため**しない**。銘柄選定は従来通り注目銘柄検出（出来高）とシグナル成立で行う
- このOOSデータ（2025-10〜2026-07）は1回使用済み。次回検証には新しい期間または新しい銘柄が必要
